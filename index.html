Tег: <!DOCTYPE html> - нужен что бы указать тип кода

Tег: <html lang="en"> - нужен что бы указать браузеру язык вашего сайта

Tег: <meta charset="UTF-8"> - указывает кодировку сайта в этом случае UTF-8

Tег: <head> - предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными

Tег: <title> - указывает названия в вкладке

Tег: <link rel="stylesheet" href="../CSS/DeskBookStyle.css"> - указывает стиль шрифта по ссылке на файл

Tег: <link rel="shortcut icon" href="../Image/favicon.ico"> - указывает иконку в вкладке по ссылке на иконку

Tег: <h1> - это заголовок и стиль заголовка (h1)

Tег: <body> - предназначен для хранения содержания сайта (контента), отображаемого в окне браузера

Tег: <div> - нужен для выведения или выделения надписи или подсказок

Tег: <p> - обзац

Tег: <i> - шрифт курсив

Tег: <footer> - подвал

Tег: <pre> - тег сохраняет все формотирования в коде и переносит его на сайт

Tег: <u> - подчеркивания текста

Tег: <a src="../"> - добавления картинки на сайт

Tег: <a href="url"> - добавления ссылки по которой перейдем по ней

Tег: <a name="название"> - добавления якаря

Tег: <a href="index.html#footer"> - Переход на главную к описанию footer
